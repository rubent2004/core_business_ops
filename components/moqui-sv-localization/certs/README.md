# Certificados DEMO

`demo-05142107981015.crt` es un certificado **autofirmado ficticio** generado solo
para que la fila `DteCertificate` del seed resuelva. El sistema corre en modo
**mock** (`mockEnabled="Y"`), por lo que este certificado **nunca se usa para
firmar ni transmitir** documentos a ningún ambiente del Ministerio de Hacienda.

No tiene valor legal ni criptográfico real. No incluir certificados reales aquí.
